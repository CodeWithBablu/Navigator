<h2>🛠️ Installation Steps:</h2>

<p>1. Get the latest snapshot</p>

```
git clone https://github.com/CodeWithBablu/Navigator.git
```

<p>2. Install NPM dependencies</p>

```
npm install
```

<p>3. Then simply start your app</p>

```
npm run dev
```

**Extra configuration**
  -  tailwindcss
